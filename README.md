# minha-pagina-pessoal
Projeto de pagina pessoal usando html e css para aprender git
## Respostas

### 1.
Commits frequentes ajudam a manter um histórico claro, facilitam o trabalho em equipe e permitem corrigir erros com segurança.

### 2.
Revert Commit: desfaz um commit criando outro.
Editar arquivo: corrige manualmente criando novo commit.

### 3.
O histórico mostra quando e o que foi alterado, ajudando a encontrar bugs e entender o problema.
